# Force Deleter

指定されたフォルダを、中身ごと強制的に削除するためのアプリケーションです。

## 概要

このアプリケーションは、ダブルクリックで実行できる単一のファイル（`QuickDeleter.exe`）です。実行すると、ローカルでWebサーバーが起動し、**自動的にブラウザで操作画面が開きます。**

CRCエラー（ディスク上のデータ破損）など、通常の削除コマンドでは失敗するようなケースでも、Windowsの強力な削除コマンド `rd /s /q` を利用して強制的に削除を試みます。

処理内容は `quick_deleter_log.txt` というファイルに記録され、コマンドの実行結果（成功またはエラー）を後から確認できます。

## 使い方

### 1. アプリケーションの実行

`dist` フォルダの中にある `QuickDeleter.exe` をダブルクリックして実行します。

初回実行時、Windows Defenderなどのセキュリティソフトによって警告が表示される場合がありますが、問題ありませんので「詳細情報」→「実行」を選択してください。

実行すると、**自動的にWebブラウザが起動し、操作画面が表示されます。**

### 2. フォルダの削除

表示されたWebページの入力欄に、削除したいフォルダの**フルパス**を入力し、「削除実行」ボタンを押してください。

処理が完了すると、Webページのログエリアに結果が表示されます。

### 3. ログの確認

`QuickDeleter.exe` と同じ場所（`dist` フォルダ内）に `quick_deleter_log.txt` というログファイルが作成され、処理の詳細が記録されます。

## トラブルシューティング

### CRCエラー（巡回冗長検査エラー）が表示された場合

これは、ハードディスクやSSD上のファイルが破損していることを示すOSレベルのエラーです。本アプリケーションは強制的な削除を試みますが、このエラーが頻発する場合は、ディスク自体に問題がある可能性があります。

その際は、以下の手順でディスクのチェックと修復を行ってください。

1.  コマンドプロンプトを**管理者として実行**します。
2.  `chkdsk D: /f /r` のように、エラーが発生したドライブ（例では `D:` ドライブ）を指定してコマンドを実行します。
3.  チェックが完了したら、再度本アプリケーションでの削除を試みてください。

## 開発者向け情報

このアプリケーションはPythonとFlaskで作成されています。

### 必要なライブラリ

```sh
pip install Flask
```

### ソースコードからの起動

```sh
cd path\to\quick_files_deleter
python app.py
```

### 実行ファイルのビルド

```sh
pip install pyinstaller
cd path\to\quick_files_deleter
pyinstaller --onefile --windowed --name QuickDeleter app.py
